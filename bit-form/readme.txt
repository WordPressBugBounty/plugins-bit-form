### Conditional Logic and Dynamic Forms

== External services ==

Bit Form connects to external services only when:
- You enable a specific integration (e.g., Zoho CRM, Mailchimp, Google Sheets)
- You use a built-in feature that requires external processing (e.g., spam protection, icon search)

Data is sent to external servers only when integrations are active (captcha/icon search are exceptions when those features are used). Enabling an integration sends form data to that service per your configuration — for example, connecting Mailchimp will send subscriber data on form submission.

For a full reference with additional details see: [External Services](https://github.com/bit-apps-pro/bit-form-frontend/blob/main/external-services.md)



## FAQ

= Can I build advanced forms with Bit Form? =
Yes. Bit Form supports conditional logic, calculations, multistep forms, conversational forms, file uploads, email templates, form entries, integrations, and advanced Pro features.

= Does Bit Form support payment forms? =
Yes. Bit Form Pro supports payment forms with PayPal, Stripe, Razorpay, and Mollie. You can create order forms, booking forms, donation forms, and service payment forms.

= Can I create multistep forms? =
Yes. Bit Form supports multistep forms, so you can split long forms into smaller steps for applications, bookings, registrations, quotes, and payment forms.

= Can users upload files through Bit Form? =
Yes. Bit Form includes a file upload field in the free version. Bit Form Pro adds advanced file upload controls for more complex form workflows.

= Does Bit Form include spam protection? =
Yes. Bit Form supports reCAPTCHA v2, reCAPTCHA v3, hCaptcha, and Cloudflare Turnstile. Bit Form Pro also includes honeypot spam protection.

= Does Bit Form Free show form entries? =
Yes. Bit Form free version shows form submissions inside WordPress, so you can view and manage entries from your dashboard. You can also disable database entry storage when needed.

= Are Bit Form integrations free? =
Yes. Bit Form includes 50+ free integrations for CRM, email marketing, automation, spreadsheets, storage, messaging, WooCommerce, webhooks, and other workflow tools.

= Does Bit Form work with page builders? =
Yes. You can use Bit Form with WordPress pages, posts, shortcodes, Gutenberg, and popular page builders that support shortcode or block-based form placement.

= Will Bit Form slow down my website? =
Bit Form is built as a lightweight form builder. For simple contact forms, it loads only the required assets, so your forms stay fast and clean.

= Can I build multi-column forms? =
Yes, Bit Form supports multi-column form layouts. You can place fields side by side, adjust the form structure, and create cleaner layouts for contact forms, booking forms, order forms, registration forms, and other advanced forms.

= Can I export/import form submission data? =
Yes, you can export form submission data from Bit Form entries for reporting, backup, or external use.

== Installation ==

1. Download the Bit Form plugin.
2. Upload the Bit Form plugin files to the `/wp-content/plugins/bit-form` directory, or from the WordPress Admin Panel, click on Plugins => Add New to install Bit Form directly.
3. Click on Upload, so you can directly upload the Bit Form zip file.
4. Use the browse button to select the plugin zip file that was downloaded, and then click on Install Now.
5. Once installed, click “Activate” to enjoy the best form building experience ever.
